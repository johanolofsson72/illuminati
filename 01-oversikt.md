# Översikt och tidslinje

Den bayerska Illuminatiorden (tyska *Illuminatenorden*, "de upplystas orden") var ett hemligt sällskap under upplysningen, grundat den **1 maj 1776** i Ingolstadt i Kurfurstendömet Bayern av **Adam Weishaupt**. Orden förespråkade förnuft, sekulär moral och mänsklig fullkomningsbarhet, och stod i skarp opposition mot vidskepelse, prästvälde och furstligt maktmissbruk. Den växte snabbt genom att infiltrera frimureriet, förbjöds genom kurfurst Karl Theodors edikt 1784–1787 och var i praktiken upplöst omkring 1790.

## Kort sammanfattning

- **Ett upplysningssällskap, inte en världskonspiration.** Den verkliga orden var liten, tyskspråkig, akademiskt präglad och kortlivad. Den moderna myten behandlas separat i [[10-historia-vs-myt]].
- **Grundare:** Adam Weishaupt (1748–1830), professor i kanonisk rätt, täcknamn *Spartacus*. Se [[02-adam-weishaupt]].
- **Ursprungligt namn:** *Bund der Perfektibilisten* (Perfektibilisternas förbund), omdöpt till *Illuminatenorden* omkring april 1778.
- **Organisatör:** Adolph Knigge (*Philo*), som från 1780 ympade ett frimurerskt gradsystem på orden och drev den explosiva tillväxten. Se [[04-organisation-och-grader]] och [[06-frimureriet]].
- **Storlek:** dokumenterat omkring 650 medlemmar i slutet av 1784; totalt cirka 1 394 identifierade medlemmar över ordens livstid. Ordens egna anspråk om ~2 500 bör betraktas som partiska. Se [[07-spridning]].
- **Fall:** krossad genom interna strider, avhopp och kurfurst Karl Theodors edikt; de beslagtagna arkiven publicerades 1787. Se [[08-forbud-och-fall]].

## Namn och begrepp

| Term | Betydelse |
|------|-----------|
| *Illuminaten / Illuminati* | "De upplysta" — medlemmarna |
| *Bund der Perfektibilisten* | Ordens första namn (1776–1778) |
| *Perfektibilism* | Läran om människans moraliska fullkomningsbarhet |
| *Minervalkirche* | Lokal cell/mötesförsamling i Minerval-graden |
| *Areopagen* | Ordens högsta råd under Weishaupt |
| *Ordensnamen* | Klassiska täcknamn (t.ex. Spartacus, Philo, Cato) |
| *Quibus Licet (Q.L.)* | Medlemmarnas obligatoriska månadsrapporter |

## Tidslinje

| År | Händelse |
|----|----------|
| **6 feb 1748** | Adam Weishaupt föds i Ingolstadt |
| **1773** | Jesuitorden upplöses av påven; Weishaupt får den lediga lärostolen i kanonisk rätt |
| **1 maj 1776** | Orden grundas som *Bund der Perfektibilisten* med fem medlemmar |
| **feb 1777** | Weishaupt går med i frimurarlogen "Prudence" i München |
| **1778** | Orden döps om till *Illuminatenorden*; ~27 medlemmar |
| **21 mars 1779** | Logen "Theodor zum guten Rat" i München tas över och fylls med illuminater |
| **1780** | Adolph Knigge (*Philo*) rekryteras; ~60 medlemmar |
| **20 jan 1782** | Knigge formaliserar det nya gradsystemet i tre klasser |
| **16 juli–sep 1782** | Frimurarkongressen i Wilhelmsbad; orden söker inflytande |
| **vår 1782** | ~300 medlemmar och kontroll över flera loger |
| **okt 1783** | Joseph von Utzschneider inleder angiveriet mot orden |
| **1 juli 1784** | Knigge lämnar orden efter bitter konflikt med Weishaupt |
| **22 juni 1784** | Karl Theodors första edikt mot hemliga sällskap |
| **2 mars 1785** | Andra ediktet, som uttryckligen namnger orden — dödsstöten i Bayern |
| **10 juli 1785** | Kuriren "Lanz" dödas av blixtnedslag (omtvistad episod) |
| **okt 1786** | Husrannsakningar hos Zwack (Landshut) och Bassus (Sandersdorf) |
| **26 mars 1787** | Regeringen publicerar *Einige Originalschriften des Illuminatenordens* |
| **16 aug 1787** | Tredje ediktet: dödsstraff för värvning till orden |
| **1787–1790** | Orden upphör som organiserad kropp |
| **1793** | J.C. Bodes död markerar ordens definitiva slut |
| **1797–98** | Robison och Barruel skapar konspirationsmyten |
| **18 nov 1830** | Weishaupt dör i Gotha under hertig Ernst II:s beskydd |
