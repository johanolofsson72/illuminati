# Moderna reproduktioner (ej original)

Denna mapp är strikt åtskild från `../` (de äkta 1700-talsbilderna). Här ligger **moderna återgivningar** av ordens emblem — tydligt märkta som sådana, så att arkivet aldrig blandar ihop dem med originalmaterial. Se avgränsningen i [[../../13-emblem-och-symboler]].

---

## Manifest över reproduktioner

| Fil | Beskrivning | Äkthetsbedömning | Användning |
|-----|-------------|------------------|-----------|
| [`../../logo1.png`](../../logo1.png) | Detaljerad graverad omgravyr av Minerval-ugglan (uggla på uppslagen bok, lagerkrans, moln, bergskuliss) | **Modern reproduktion.** Ikonografiskt trogen 1788 års design, men inte ett periodtryck. | Repo-logo / README-header, tydligt märkt som reproduktion |
| [`../../logo2.png`](../../logo2.png) | Ren linjeteckning / "målarbok"-stil, sannolikt AI-genererad | **Modern nyteckning**, längst från originalet. | Reservlogo; används ej som historiskt belägg |

Bägge filerna ligger i repo-roten som `logo1.png` / `logo2.png` (de tjänar som projektets grafiska identitet) och är **medvetet åtskilda** från det äkta periodmaterialet i [`../`](../). Ingen av dem används som historisk källa.

## Not om filerna

Bilderna som skickades in i chatten kunde jag **inte** automatiskt spara till disk (de kom som bifogade bilder, inte som filer på filsystemet). För att lägga in reproduktion nr 1:

1. Släpp filen i denna mapp under namnet `minerval-uggla-omgravyr.png`, **eller**
2. Peka mig till en källa/URL så hämtar jag den (och kontrollerar licens).

Oavsett vilket bevaras märkningen "modern reproduktion, ej original" så att gränsen mot [[../minerval-insignia-1788]] och [[../illuminati-emblem-uggla-1788]] hålls knivskarp.
