# Facsimiler — kompletta sidbilder (primärtryck)

Här ligger de **fullständiga originaltrycken som facsimil-PDF** (inskannade sidbilder). Det är i dessa de kompletta ritualtexterna finns — inte som ren text utan som bild. De renskrivs (transkriberas) via **Claude-visionläsning** av sidorna, vilket ger högre kvalitet än den maskinella djvu-OCR:en. Se resultat i [[../primarkallor/07-ritualer-hogupplost]].

## Metod: transkription via Claude-vision

De kompletta ritualtexterna **finns som bild** i dessa PDF:er. För att göra dem till läsbar, översättbar text läses sidorna med Claude-modellens bildläsning (multimodalt) — inte en separat OCR-tjänst. Fördelen mot maskinell OCR: modellen använder språklig kontext och läser fraktur (långt ſ, ligaturer) betydligt renare. Resultaten samlas i [[../primarkallor/07-ritualer-hogupplost]], sida för sida, med sidhänvisning till facsimilen.

> **Så refereras en sida:** `Faber 1788, s. 63` = tryckt sidnummer i facsimilen (ej PDF-sidnummer, som har ett förskott för titelblad/försättsblad på ca 5 sidor).
